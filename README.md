# Artistic Reading Journey

An immersive web experience that transforms books and paintings into an interactive digital journey with sounds and words.

## Features

- Page-by-page navigation through the story
- Visual art accompanying each page
- Audio elements for enhanced immersion
- Responsive design with artistic styling

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Building for Production

```bash
npm run build
```

## Technologies Used

- React
- TypeScript
- Vite
- CSS for styling

## Customization

- Add your own images to the `public` folder and update the `imageUrl` in `src/App.tsx`
- Add audio files to the `public` folder and update the `audioUrl`
- Modify the `pages` array in `src/App.tsx` to add more content
